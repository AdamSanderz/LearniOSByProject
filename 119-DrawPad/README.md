DrawPad
----------

DrawPad 是一个简单的绘图应用。
知识点包括：
- Draw lines and strokes using Quartz2D.
- Use multiple colors.
- Set the brush stroke width and opacity.
- Create an eraser.
- Create a custom RGB color selector.
- Share your drawing!
