# LearniOSByProject
-------

> - 学习iOS的一些小项目，主要是Swift，少部分OC。  
> - 以数字为目录只是为了记录项目数和排序，没有特殊意义。
> - 下载单个项目可以参考 [DownGit：怎么下载github中某个文件夹？](http://www.jianshu.com/p/ca46d0fa69ef)

本来是自己学习iOS时的一些非常简单小例子，小项目，或者我个人博客[AndyRon's Blog](http://andyron.com)文章的代码。没有按照什么路线学习，就是学到什么就是什么了，看上去特别乱，然后看到道长前辈的项目[Swift-30-Projects](https://github.com/soapyigu/Swift-30-Projects)，就整理、删选一下，这样看着清晰一点，找起来也方便。
目的也是为了之后遇到某个知识点能快速找到例子代码！

以为数字开始的目录没有什么特殊含义，只是先后顺序，方便寻找
